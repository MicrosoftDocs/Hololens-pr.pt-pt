# <a name="hololens-2-capabilities-and-solutions"></a>HoloLens 2 capacidades e soluções

## <a name="why-hololens"></a>Por que HoloLens?

Graças às aplicações HoloLens 2 e Da Realidade Mista, agora os trabalhadores podem colaborar sem limites, agir com precisão e aumentar a produtividade.

Colaborar sem limites e agir com precisão para aumentar a produtividade dos colaboradores com aplicações de realidade mista no HoloLens 2. Mantenha-se em frente, mãos livres mais tempo e mais confortavelmente com comandos de voz incorporados, rastreio de olhos e ancoragem mundial para foco contínuo em completar tarefas com segurança sem erros. Ligação com colegas remotos em tempo real e trabalhem em conjunto através de uma extensa tela holográfica sobreposta no seu ambiente físico para resolver rapidamente os problemas no ponto de trabalho. Realize o retorno do investimento (ROI) logo a contar da caixa com um ecossistema robusto de aplicações que são suportadas com a segurança, fiabilidade e escalabilidade da Microsoft.  

Esperamos que este artigo o inspire a pensar em todas as formas pelas quais a realidade mista (MR) irá beneficiar o seu negócio.

## <a name="hololens-2-capabilities"></a>HoloLens 2 capacidades

O que torna o HoloLens 2 tão poderoso?

| Funcionalidade | Descrição |
|---------|-------------|
| Ancoragem mundial | Hologramas ancorados permanecem precisamente no lugar. HoloLens 2 entende o seu espaço de trabalho. Assim, o conteúdo digital persiste ao longo do tempo – ancorado em objetos ou superfícies onde trabalha. |
| Rastreio de mãos | Toque, agarre e mova hologramas de formas que se sintam naturais. HoloLens 2 adapta-se às suas mãos para uma nova satisfação nas suas interações. |
| Rastreio de olhos | Desfrute de um novo nível de contexto e compreensão humana. HoloLens 2 compreende precisamente onde está a olhar, para que possa entender a sua intenção e adaptar os hologramas aos seus olhos em tempo real. |
| Voz ativada | Os comandos de voz incorporados permitem-lhe navegar e operar rapidamente HoloLens 2 quando as suas mãos estão ocupadas com uma tarefa. |
| Ergonomia | HoloLens 2 é leve (3,28 kg) que inclui um sistema de ajuste de marcação para suportar o uso prolongado. |
| Grande FoV | Expanda a sua tela holográfica com ecrãs de alta resolução e grande campo de visão. |
| Sem amarras | Mova-se livremente, sem fios ou pacotes externos para conseguir o seu caminho no trabalho. |
| Azure-powered | Transmita conteúdo 3D de alta fidelidade que pode ser ancorado a uma localização e/ou objeto que persiste entre utilizadores com serviços de realidade mista Azure.
| Captura de realidade mista | Documente uma experiência como uma foto ou um vídeo para partilhar com os outros em tempo real. |
| Windows Hello para empresas | A autenticação biométrica baseada na íris leva-o rapidamente e de forma segura ao fluxo de trabalho. |
| Windows Autopilot | Crie e prepare os serviços para HoloLens 2 para que estejam prontos a usar fora da caixa através de trabalhos distribuídos. |
| Atualizações do OS | Mantenha-se seguro com atualizações mensais de manutenção e aproveite as novas capacidades de produtividade e gerenciabilidade em lançamentos bianuais. |
| Gerir facilmente dispositivos | Gerencie vários dispositivos HoloLens 2 simultaneamente utilizando soluções como Microsoft Intune, VMware Workspace One, MobileIron e muito mais. |
| Operar em ambientes regulados | HoloLens 2 tem um extenso portfólio de dispositivos que suporta ambientes altamente regulados, incluindo ambientes designados iso classe 5.0 e UL Classe I, Divisão 2. |

## <a name="managing-hololens-2-in-your-organization"></a>Gerir HoloLens 2 na sua organização

HoloLens 2 inclui funcionalidades que facilitam a gestão e utilização de dispositivos HoloLens. Algumas funcionalidades estão incluídas no dispositivo, enquanto outras podem ser ativadas pela [Mobile Device Management (MDM) para HoloLens](hololens-mdm-configure.md) ou através de [Pacotes de Provisionamento](hololens-provisioning.md) utilizando [Windows Designer de Configuração](app-deploy-provisioning-package.md#setup).

| Eu quero... | Solução | Description |  
|---------| ------------|------------|
Gerir como os meus utilizadores finais se inscrevem | [**Identidade**](hololens-identity.md) | HoloLens 2 suporta vários tipos de identidades de utilizador - Azure Ative Directory (AAD), Microsoft Account (MSA) e contas locais.  |
| Criptografe os dados do utilizador | [**Segurança de dados**](security-encryption-data-protection.md) | A encriptação de dados BitLocker está ativada no HoloLens 2 para fornecer o mesmo nível de proteção de segurança que qualquer outro dispositivo Windows. | 
Gerir dispositivos HoloLens na minha organização | [**Gestão de Dispositivos Móveis**](hololens-mdm-configure.md) | Gerencie as definições, selecione apps para instalar e definir configurações de segurança adaptadas à necessidade da sua organização em vários dispositivos HoloLens 2 a partir de uma localização central. | 
|Minimizar o tempo de configuração para novos utilizadores e dispositivos | [**Piloto automático**](hololens2-autopilot.md) | Configure a experiência fora de caixa (OOBE) em Microsoft Endpoint Manager e permita que os utilizadores finais preparem dispositivos para uso empresarial com pouca ou nenhuma interação. |  
| Controle as atualizações do SISTEMA para os meus dispositivos | [**Windows Update para Empresas**](hololens-updates.md#managing-updates-by-using-windows-update-for-business) | Windows A Update for Business fornece atualizações controladas do sistema operativo aos dispositivos e suporte para o canal de manutenção a longo prazo. |  
| Permitir o descarregamento de aplicações específicas e LOB |[**Gestão de Aplicações**](app-deploy-overview.md) | Escolha como distribuir e controlar aplicativos para grupos selecionados de utilizadores HoloLens 2. | 
| Mostrar ou esconder aplicativos específicos no seu menu inicial |[**Modo quiosque**](hololens-kiosk.md) | Configurar HoloLens 2 para funcionar como um dispositivo de uso fixo para uso em demos de aplicações ou aplicações empresariais dedicadas. 
| Proteja o meu ambiente bloqueando apps | [**WDAC**](windows-defender-application-control-wdac.md) | Windows Defender O Controlo de Aplicações (WDAC) bloqueia o lançamento de aplicações e processos pelo utilizador do dispositivo.
| Gerir a segurança do dispositivo com regras para apps e processos | [**Políticas (CSPs)**](hololens-csp-policy-overview.md) | Os administradores de TI podem definir e implementar definições de políticas utilizando uma lista existente de CSPs de política apoiada no HoloLens 2. |  
| Gerir como um dispositivo se conecta à internet | [**Rede e Conectividade**](hololens-certificates-network.md) | Utilize a autenticação baseada em certificados para aceder a Wi-Fi, VPNs ou recursos internos. | 
| Partilhe o dispositivo com vários utilizadores | [**Display personalizado automaticamente**](hololens-calibration.md#auto-eye-position-support) | HoloLens 2 ecrãs são automaticamente ajustados com a Posição Auto Eye (AEP), eliminando a necessidade de executar um processo de calibração manual quando o dispositivo é [partilhado entre os utilizadores](hololens-multiple-users.md). |

Conheça os [Requisitos de Licenciamento](hololens-licenses-requirements.md) para as soluções acima referidas.
